# hand-tear-ahooks

学习研究[ahooks](https://ahooks.js.org/zh-CN/hooks/async)，最终实现一个 ahooks 的功能

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
