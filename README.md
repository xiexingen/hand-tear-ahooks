# hand-tear-ahooks

_每天花一点空闲时间来_ 手撕[ahooks 源码](https://ahooks.js.org/zh-CN/hooks/async)，最终实现一个 ahooks 的功能

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
